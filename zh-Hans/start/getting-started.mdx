LiftAuth 是一个为新加坡电梯服务公司打造的电梯维护管理平台。它帮助您安排服务访问、记录已完成的工作、跟踪故障并发送发票——所有这些都集中在一个地方。

<CardGroup cols={3}>
  <Card title="Admins" icon="monitor">
    在网页控制台上规划和监督运营的办公室员工。
  </Card>

  <Card title="Technicians" icon="wrench">
    使用移动应用在现场执行任务的现场工作人员。
  </Card>

  <Card title="Building Managers" icon="building">
    负责一栋建筑。通过电子邮件接收需要签署的记录——无需登录。
  </Card>
</CardGroup>
