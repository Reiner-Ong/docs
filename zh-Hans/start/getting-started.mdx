LiftAuth 是为新加坡电梯维修公司构建的电梯维护管理平台。它帮助您安排服务访问、记录完成的工作、跟踪故障和发送发票 — 所有功能集于一处。

<CardGroup cols={3}>
  <Card title="管理员" icon="monitor">
    通过 Web 仪表板规划和监督运营的办公室员工。\
  </Card>

  <Card title="技术员" icon="wrench">
    使用移动应用程序在现场执行工作的实地工作人员。\
    \
  </Card>

  <Card title="建筑经理" icon="building">
    负责一栋建筑。通过电子邮件接收要签字的记录 — 无需登录。
  </Card>
</CardGroup>

## 基于角色的访问

<div style={{ marginTop: "-1rem", marginBottom: "-1rem" }}>
  | 角色           | 他们可以做什么                                                                                         |
  | -------------- | -------------------------------------------------------------------------------------------------------- |
  | **Owner**      | 完全访问 — 管理团队、计费、发票和所有设置。<br />还具有技术员访问权限。     |
  | **Admin**      | 管理工作、发票和团队成员。无法更改计费。<br />还具有技术员访问权限。         |
  | **Technician** | 仅限移动应用程序。只能看到分配给他们的工作。无法访问发票、付款或团队设置。 |
</div>

<Tip>
  每个公司从一个 Owner 开始。然后 Owner 可以从 Team 页面邀请其他管理员和技术员。
</Tip>
