---
title: "获取签名"
description: "完成任务后如何获取 building manager 的签名。"
---

您提交工作后,building manager 必须在记录上签字。您有两个选项。

## 选项 1 — 在您的手机上签字

如果 building manager 在现场,他们可以直接在您的手机上签字。

<Steps>
  <Step title="点击 Sign on device">
    在签名屏幕上,点击 **Sign on device**。
  </Step>
  <Step title="将手机交给 building manager">
    他们输入姓名并在屏幕上绘制签名。
  </Step>
  <Step title="点击 Submit signature">
    记录已签署,任务转为 **Signed**。
  </Step>
</Steps>

## 选项 2 — 发送签署链接

如果 building manager 不在现场,您可以向他们发送链接以远程签字。

<Steps>
  <Step title="点击 Signature link 下的 Share">
    在签名屏幕上,点击 **Share** 以通过电子邮件或短信发送链接,或点击链接以手动复制。
  </Step>
  <Step title="点击 I've shared the link">
    点击 **Share Link**。
  </Step>
</Steps>

building manager 将收到链接并可以在自己的设备上签字。一旦他们完成,任务将自动转为 **Signed**。

<Tip>
  如果 building manager 未收到链接,请要求您的 admin 从 [Records](/admin/records#sending-a-signing-link) 页面重新发送。
</Tip>
