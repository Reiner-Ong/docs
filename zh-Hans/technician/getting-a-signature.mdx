---
title: "获取签名"
description: "完成工单后如何获取 building manager 的签名。"
---

提交工作后,building manager 必须对记录签字。您有两个选项。

## 选项 1 — 在您的手机上签字

如果 building manager 在现场,他们可以直接在您的手机上签字。

<Steps>
  <Step title="点击 Sign on device">
    在签字屏幕上,点击 **Sign on device**。
  </Step>
  <Step title="将您的手机交给 building manager">
    他们输入姓名并在屏幕上绘制签名。
  </Step>
  <Step title="点击 Submit signature">
    记录已签字,工单进入 **Signed** 状态。
  </Step>
</Steps>

## 选项 2 — 发送签字链接

如果 building manager 不在现场,您可以向他们发送一个链接以便远程签字。

<Steps>
  <Step title="在 Signature link 下点击 Share">
    在签字屏幕上,点击 **Share** 以通过电子邮件或短信发送链接,或点击链接手动复制。
  </Step>
  <Step title="点击 I've shared the link">
    点击 **Share Link**。
  </Step>
</Steps>

building manager 将收到链接,并可以在他们自己的设备上签字。一旦他们完成,工单将自动进入 **Signed** 状态。

<Tip>
  如果 building manager 没有收到链接,请要求您的 admin 从 [Records](/zh-Hans/admin/records#sending-a-signing-link) 页面重新发送。
</Tip>
