---
title: "提出 Issue"
description: "如何报告您在任务之外发现的电梯问题。"
---

如果您发现电梯有问题——即使它不属于您预定的任务——您也可以直接从应用报告。您的 admin 会在他们的 Inbox 中看到它。

## 如何提出 Issue

<Steps>
  <Step title="打开电梯">
    前往 **Lifts** 选项卡,然后点击您发现问题的电梯。
  </Step>
  <Step title="点击 Report Issue">
    在电梯的详细信息页面,点击 **Report Issue**。
  </Step>
  <Step title="选择严重性">
    选择问题的严重程度:

    | 严重性 | 何时使用 |
    | --- | --- |
    | **Critical** | 电梯有安全风险——立即停止使用 |
    | **Major** | 电梯仍可工作,但存在运行问题 |
    | **Minor** | 外观或轻微问题,不影响运行 |
  </Step>
  <Step title="描述问题">
    简要描述您发现的内容。要具体——包括您看到、听到或感觉到的内容。
  </Step>
  <Step title="添加照片(推荐)">
    点击相机图标以附加照片。照片可以帮助您的 admin 在无需访问现场的情况下理解问题。
  </Step>
  <Step title="提交">
    点击 **Submit**。issue 已保存,您的 admin 会收到通知。
  </Step>
</Steps>

<Warning>
  如果您选择 **Critical**,您的 admin 将立即收到警报。如果您认为电梯不安全,请勿让其继续服务。
</Warning>
