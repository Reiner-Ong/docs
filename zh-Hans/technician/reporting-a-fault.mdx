---
title: "提出 Issue"
description: "如何报告您在工单之外发现的电梯问题。"
---

如果您注意到电梯有问题 — 即使这不是您预定工单的一部分 — 您可以直接从应用报告它。您的 admin 将在他们的 Inbox 中看到它。

## 如何提出 Issue

<Steps>
  <Step title="打开电梯">
    前往 **Lifts** 选项卡,点击您发现问题的电梯。
  </Step>
  <Step title="点击 Report Issue">
    在电梯的详情页面上,点击 **Report Issue**。
  </Step>
  <Step title="选择严重程度">
    选择问题有多严重:

    | 严重程度 | 何时使用 |
    | --- | --- |
    | **Critical** | 电梯存在安全风险 — 立即停止使用 |
    | **Major** | 电梯仍能工作但有运行问题 |
    | **Minor** | 不影响运行的外观或轻微问题 |
  </Step>
  <Step title="描述 issue">
    简短描述您发现的问题。要具体 — 包括您看到、听到或感觉到的内容。
  </Step>
  <Step title="添加照片(推荐)">
    点击相机图标以附加照片。照片有助于您的 admin 在无需访问现场的情况下理解问题。
  </Step>
  <Step title="提交">
    点击 **Submit**。issue 已保存,您的 admin 已收到通知。
  </Step>
</Steps>

<Warning>
  如果您选择 **Critical**,您的 admin 将立即收到警报。如果您认为电梯不安全,请不要让它继续投入使用。
</Warning>
