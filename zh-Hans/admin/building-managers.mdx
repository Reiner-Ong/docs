## 添加新的 building manager

<Steps>
  <Step title="前往 Building Managers">
    点击左侧栏中的 **Building Managers**。
  </Step>
  <Step title="点击 Add">
    点击右上角的 **Add building manager** 按钮。
  </Step>
  <Step title="输入公司详细信息">
    填写公司名称和街道地址。
  </Step>
  <Step title="添加联系人">
    输入联系人的姓名、电子邮件地址和电话号码。您可以为一个 Building Manager 添加多个联系人。
  </Step>
  <Step title="保存">
    点击 **Save**。building manager 现在出现在您的列表中。
  </Step>
</Steps>

<Warning>
  请仔细检查电子邮件地址——发票和签署请求将发送至此。
</Warning>
