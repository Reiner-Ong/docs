maintenance record 是技术员完成任务时生成的报告。它记录了所做的工作,并作为 Building Manager 在任务完成后收到的报告。

<Warning>
  一旦记录创建,就无法更改。
</Warning>

## 如何签署记录 {#sending-a-signing-link}

如果 building manager 不在现场签字,作为 admin 在场外为他们获取签署链接有两种方式。

<CardGroup cols={2}>
  <Card icon="wallet-cards">
    ## 直接发送

    LiftAuth 自动将链接通过电子邮件发送给 building manager 的联系人。打开记录并点击 **Send signing link** ——无需进一步输入。
  </Card>

  <Card icon="link">
    ## 复制链接

    LiftAuth 生成链接并将其复制到您的剪贴板。然后您可以通过 WhatsApp、短信或任何其他方式自行分享。
  </Card>
</CardGroup>

一旦 building manager 点击链接并签字,记录状态会自动更新为 **Signed**,任务出现在您的 Inbox 中,准备关闭。他们那边不需要 LiftAuth 帐户。
